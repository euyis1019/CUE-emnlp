\section{Conclusion}
\label{sec:conclusion}

% 中文: 我们提出了一个逐层追踪 LLM 代码推理的 dual diagnostic framework，揭示了一个从 brewing 到 resolution 的结构化内部过程，以及四种具有因果支持的 resolution outcome。不同 code primitive 诱导出不同的 outcome fingerprint，表明 code understanding 不是单一能力而是一组具有不同瓶颈的内部子过程。
We proposed a dual diagnostic framework that traces code reasoning inside LLMs layer by layer, revealing a structured \textbf{brewing} process---answer information becomes externally readable before it becomes self-decodable---followed by four causally validated \textbf{resolution outcomes}. Different code primitives induce distinct outcome fingerprints, showing that code understanding is not a monolithic capability but a family of internal sub-processes with different bottlenecks. Because these states leave ground-truth-free traces in the model's own distributions, the framework opens paths toward failure diagnosis, confidence estimation, and adaptive computation for code LLMs.
