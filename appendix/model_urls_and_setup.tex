\section{Model URLs and Setup}
\label{app:model_urls_and_setup}

All models used in this work are publicly available on Hugging Face. We download the original pre-trained weights without further fine-tuning or quantization.

\paragraph{Primary model family (anchor + scaling).}
Our main analyses use the Qwen2.5-Coder series \citep{hui2024qwen25coder} spanning five scales, paired with the general-purpose Qwen2.5 counterparts \citep{yang2024qwen25} of the same sizes:

\begin{itemize}[nosep,leftmargin=1.5em]
  \item \texttt{Qwen/Qwen2.5-Coder-0.5B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-Coder-0.5B}
  \item \texttt{Qwen/Qwen2.5-Coder-1.5B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-Coder-1.5B}
  \item \texttt{Qwen/Qwen2.5-Coder-3B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-Coder-3B}
  \item \texttt{Qwen/Qwen2.5-Coder-7B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-Coder-7B}
  \item \texttt{Qwen/Qwen2.5-Coder-14B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-Coder-14B}
  \item \texttt{Qwen/Qwen2.5-0.5B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-0.5B}
  \item \texttt{Qwen/Qwen2.5-1.5B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-1.5B}
  \item \texttt{Qwen/Qwen2.5-3B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-3B}
  \item \texttt{Qwen/Qwen2.5-7B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-7B}
  \item \texttt{Qwen/Qwen2.5-14B} \quad\url{https://huggingface.co/Qwen/Qwen2.5-14B}
\end{itemize}
